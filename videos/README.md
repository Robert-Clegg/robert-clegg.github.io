# Pathogenika Demo Videos

Drop `.mp4` gameplay recordings here. They'll be served via GitHub Pages.

## Usage
Access via URL parameter: `https://robert-clegg.github.io/pathogenika/?v=FILENAME`

Where FILENAME is the file name without `.mp4` extension.

## Example
- File: `commander-demo.mp4`
- URL: `https://robert-clegg.github.io/pathogenika/?v=commander-demo`

## Limits
- GitHub repos: 100MB per file, ~1GB total recommended
- 3-min match at 720p ≈ 30-50MB — fits fine
- For larger/more videos, migrate to YouTube (`?yt=VIDEO_ID`)
